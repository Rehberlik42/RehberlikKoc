2. Teknoloji Yığını

Çekirdek

  • Next.js 16.2.6 — App Router, React Server Components (RSC)
  • React 19.2.4
  • TypeScript 5 (strict mode)
  • Tailwind CSS v4
  • Geist Sans / Geist Mono — next/font/google

Backend / Veri

  • Supabase — Auth + Postgres + RLS
  • @supabase/ssr 0.10.3 — cookie tabanlı SSR oturum yönetimi
  • @supabase/supabase-js 2.106.2
  • REST API yok — tüm veri işlemleri doğrudan Supabase client üzerinden

UI kütüphaneleri

  • lucide-react — ikonlar
  • recharts — deneme grafikleri
  • @dnd-kit — öğretmen randevu Kanban tahtası
  • html2canvas-pro + jspdf — istemci tarafı PDF export

Next.js 16 farkları

  • middleware.ts yerine kök dizinde proxy.ts kullanılıyor
  • AGENTS.md: Next.js 16 breaking change'ler var; kod yazmadan önce node_modules/next/dist/docs/ okunmalı
  • Dev script: next dev --webpack

6. Kimlik Doğrulama ve Oturum Akışı

6.1 Kayıt (Signup)

Landing sayfasındaki AuthModal: signUp({ email, password, options: { data: { role: "student"|"teacher" } } })

  • Rol auth user_metadata.role alanına yazılıyor
  • profiles tablosuna satır oluşturan DB trigger varsayılıyor (repoda yok)
  • Kayıt sonrası e-posta doğrulama mesajı gösteriliyor

6.2 Giriş

signInWithPassword → başarılı ise /dashboard redirect

6.3 proxy.ts

  1. Cookie'lerden Supabase oturumunu yeniler
  2. getClaims() kullanır (JWT yerel doğrulama, network yok)
  3. Kullanıcı yoksa ve path /dashboard/* ise → / redirect

6.4 dashboard/layout.tsx

  1. getUser() — sunucu tarafında doğrulama
  2. profiles tablosundan profil çeker
  3. Profil yoksa → / redirect
  4. DashboardShell ile sarmalar

6.5 Rol yönlendirme

  • role === "teacher" | "admin" → /dashboard/teacher
  • Diğer → /dashboard/student

6.6 Üç Supabase client

Kritik: Şema kaynağı uzak Supabase projesi. Repoda sadece guidance_contents migration'ı var. Generated TypeScript tipleri yok.

id (UUID), full_name, role (student|teacher|admin), avatar_url, grade, school, phone, bio

8.6 topic_progress

8.7 resources

type: youtube | book | document | website; is_approved; view_count; subject_id; topic_id

8.8 psychological_tests

type: motivation | anxiety | focus | burnout | general; questions JSONB (şema lib/tests.ts'de)

8.9 test_results

student_id, test_id, answers (JSONB), score, interpretation, taken_at

8.10 appointments

status: pending | confirmed | completed | cancelled; notes; session_notes

8.11 guidance_contents

content_type: blog | video | pdf; target_exam: YKS | LGS | KPSS | ARA_SINIF

11. Bileşen Organizasyon Pattern'leri

  • Server Page + Client Island: page.tsx async RSC fetch → ClientComponent props
  • Route-colocated _components/: Her feature kendi klasöründe
  • dynamic = "force-dynamic": Veri değişken sayfalarda cache bypass
  • Supabase join normalizasyonu: Array.isArray join sonuçları düzeltilir
  • PDF Export: wrapper div id + pdf-export-hide class + PdfExportButton
